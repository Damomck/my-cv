HTML ELEMENTS

Anchor < a >

Used with href in order to create a hyperlink reference "button" which when clicked, will take the user to a particular webpage, file, email address, another location on the same page, etc.

Line Break < br >

Used to end the line and start a new one. Self closing.

Emphasis < e >

Same structural result to text as italics (see below), but emphasis additionally conveys function/meaning to the user

Form < form >

Multi-functional element that allows for interactive controls depending on the input type selected eg < input type = "submit" > creates a button to send data via a pre-chosen email delivery system
Common Usage: < form action="mailto:???@???.com" method="post" enctype="text/plain" >
< label >Name< /label >
< input type = "????" >
< /form >

Headings < h1 > to < h6 >

Heading sizes from h1 to h6, 1-3 larger than default text and all automatically bold. No need to place a paragraph < p > or line break < b > after since it is automatically padded.

Italics < i >

Used purely to structurally change text to italics

Paragraph < p >

Begin text on a new line, spaced further than a break
